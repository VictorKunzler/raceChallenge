## Setup
**1** - Set your database config at /config/config.json

**2** - Run the following commands
```
> npm install
> npm run migrate
```

**3** - To setup tests run the following commands
```
> npm run migrate:test
```

## Running
Run project with `npm start`
Run tests with `npm test`
